# 文具歸還管理系統

直接開啟 `index.html` 即可使用，不需要安裝。

## 功能

- 系統不包含任何預載人員資料，可安全部署到 GitHub
- 上傳原始 CSV 後，自動載入「09-借用類」及「10-借用類(二手品$0元)」且狀態為「已完成」的資料
- 可隨時上傳最新 CSV 更新借用清單，既有歸還進度不會被重置
- 依姓名、部門、物品搜尋
- 建立文具歸還案件
- 分開記錄離職日及每筆物品的實際歸還日期
- 支援全數歸還、部分歸還及遺失
- 自動保存在目前瀏覽器
- 匯出歸還紀錄 CSV、列印歸還清冊

## 資料保存

原始借用資料與歸還進度都只儲存在目前瀏覽器的 localStorage，不會寫入 GitHub。若更換瀏覽器或清除網站資料，紀錄會消失，請定期使用「備份進度」。
